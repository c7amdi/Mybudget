# **App Name**: BudgetWise

## Core Features:

- Expense Tracking: Log expenses with details like category, amount, and date.
- Income Tracking: Log income with details like source, amount, and date.
- Category Management: Create, edit, and delete expense and income categories.
- Currency Management: Add and manage multiple currencies for transactions.
- Account Management: Add and manage different types of accounts (e.g., cash, bank, credit card).
- Detailed Reporting: Visualize income and expense data through interactive charts and graphs.
- Translation Tool: Provide offline translations between English, French and Arabic.

## Style Guidelines:

- Primary color: Deep teal (#008080) for trustworthiness and reliability.
- Background color: Light cyan (#E0FFFF), a very desaturated teal for a clean, calm backdrop.
- Accent color: Forest green (#228B22) to highlight key actions and categories.
- Font pairing: 'Poppins' (sans-serif) for headlines, 'PT Sans' (sans-serif) for body text.
- Use simple, intuitive icons for categories and actions.
- Employ a clean, modern layout with clear separation of sections for easy navigation.
- Incorporate subtle animations and transitions to enhance user experience.
